封装一个包含私有和公共特性的类
---
```js
class Package {
       constructor (animal){
           this.animal = animal
       }
       zoo (animal) {
           console.log('this is'+ this.animal)
       }
       static private () {
           console.log('我是私有方法')
       }
   }
 
   let newPackage = new Package('大象')
   newPackage.zoo()
   newPackage.private() // 报错
```
